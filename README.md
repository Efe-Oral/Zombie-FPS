## About

Zombie FPS is a single-player horror FPS game set in a dark, nighttime environment. Armed with a gun and a flashlight with limited battery, you must survive against waves of zombie enemies that pursue you using AI-driven pathfinding. The game features two difficulty modes across its releases, with the second version offering a significantly harder challenge.

## Features

- **First-Person Shooting** — Aim and fire to take down incoming zombies
- **Flashlight & Battery System** — Manage your flashlight battery as you navigate through darkness
- **AI-Driven Enemies** — Zombies use NavMesh pathfinding to hunt the player
- **Bunker Environment** — Explore a bunker area complete with doors and interior spaces
- **Terrain & Atmosphere** — Hand-crafted terrain with a dark skybox, dynamic lighting, and post-processing effects for a horror atmosphere
- **Occlusion Culling** — Optimized rendering performance
- **Custom Shaders** — ShaderLab and HLSL shaders for visual effects

## Built With

- **Engine:** [Unity 2020.3.35f1](https://unity.com/releases/editor/whats-new/2020.3.35) (LTS)
- **Language:** C#
- **AI Navigation:** Unity NavMesh
- **Rendering:** Post-processing stack, occlusion culling
